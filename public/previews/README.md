## Formato Recomendado
- **Formato**: JPG o PNG
- **Tamaño**: 120x120px (cuadrado)
- **Peso**: < 50KB por imagen
- **Fondo**: Transparente (PNG) o que combine con el tema oscuro

## Imagen por Defecto
Si no tienes una imagen para alguna plantilla, se usará un placeholder con el ícono de la plantilla.
